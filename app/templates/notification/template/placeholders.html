Event

Use these placeholders to insert event details into your notification templates.

  • Title: {{ '{{event.title}}' }} - The event name
  • Description: {{ '{{event.description}}' }} - Event details and description
  • Location: {{ '{{event.location}}' }} - Where the event takes place
  • Start at: {{ '{{event.start_at}}' }} - Full date and time when event begins
  • End at: {{ '{{event.end_at}}' }} - Full date and time when event ends
  • Start date: {{ '{{event.start_date}}' }} - Only the start date
  • Start time: {{ '{{event.start_time}}' }} - Only the start time
  • End date: {{ '{{event.end_date}}' }} - Only the end date
  • End time: {{ '{{event.end_time}}' }} - Only the end time
  • Duration hours: {{ '{{event.duration_hours}}' }} - How long the event lasts in hours
  • Participants count: {{ '{{event.participants_count}}' }} - Number of people attending
  • Participant names: {{ '{{event_participants_names}}' }} - Comma-separated list of attendees
  • Participant emails: {{ '{{event_participants_emails}}' }} - Comma-separated email addresses
  • Google calendar: {{ '{{event_calendar_google}}' }} - Link to add event to Google Calendar
  • Outlook calendar: {{ '{{event_calendar_outlook}}' }} - Link to add event to Outlook Calendar
Current date

Insert current date and time information into your templates.

  • Date: {{ '{{current.date}}' }} - Today's date
  • Time: {{ '{{current.time}}' }} - Current time of day
  • Month: {{ '{{current.month}}' }} - Current month name
  • Month number: {{ '{{current.month_number}}' }} - Month as number (1-12)
  • Year: {{ '{{current.year}}' }} - Current year
  • Previous month: {{ '{{previous.month}}' }} - Last month's name
  • Previous month number: {{ '{{previous.month_number}}' }} - Last month as number
  • Next month: {{ '{{next.month}}' }} - Next month's name
  • Next month number: {{ '{{next.month_number}}' }} - Next month as number
Birthdays

  • This week: {{ '{{birthdays_this_week}}' }} - List of birthdays this week
